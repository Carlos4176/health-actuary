# 健康小结（给长辈）

## 1. 先说结论（安抚+鼓励，3句话以内）  
您的整体健康状况目前仍在正常范围内，很多指标都还没“踩线”，这是很好的基础！  
不过近三四年有几项数值悄悄往上走，像是血压、血糖和血脂，值得我们多留点心。  
别担心，这些变化大多和生活习惯有关，调整得早、改得巧，完全有机会稳住甚至改善！

## 2. 哪些指标要留意（最多5项）  
- **血压（收缩压和舒张压）**：这几年每年都在慢慢升高，现在舒张压刚好卡在正常上限（89 mmHg），再高一点就超了，需要关注。  
- **空腹血糖**：从2022年的4.87升到今年的5.94，虽然还在正常范围（<6.1），但趋势偏上，接近“提醒线”。  
- **甘油三酯和低密度脂蛋白（“坏”胆固醇）**：这两项都已略高于参考值，尤其是甘油三酯连续三年上升，可能和饮食、运动有关。  
- **高密度脂蛋白（“好”胆固醇）**：今年首次低于1.0，这个“保护性”指标偏低，对血管不太有利。  
- **尿酸和肝酶（AST）**：尿酸快到上限（420），AST也在逐年缓升，虽未超标，但提示代谢和肝脏负担可能在增加。

## 3. 生活习惯小建议（不超过8条）  
1. **晚餐少放一勺油、一块肥肉**：减少动物脂肪和油炸食品，有助于降甘油三酯和“坏”胆固醇。  
2. **每天快走20–30分钟**：比如晚饭后绕小区走两圈，能温和帮助控血糖、提“好”胆固醇。  
3. **主食里加点粗粮**：把白米饭换成一半糙米、燕麦或杂豆，血糖会更平稳。  
4. **喝水比以前多一点**：每天喝够6–8杯温水，有助尿酸排出，也减轻肝脏负担。  
5. **少吃内脏、浓汤和海鲜汤**：这些嘌呤高，容易推高尿酸。  
6. **尽量不熬夜、睡够7小时**：休息好，身体代谢更顺畅，心率和血压也会更稳。  
7. **情绪放松，少生闷气**：紧张焦虑会让血压和心率悄悄升高，听听戏、聊聊天都是好方法。  
8. **定期在家量血压**：每周测2–3次（早上起床后安静时），记录下来，比单次体检更真实。

## 4. 复查与就医建议  
建议您下次体检前，先整理好这几年的报告，带着一起去看**家庭医生或内科医生**，特别关注：  
- 如果在家自测血压多次超过135/85 mmHg；  
- 或出现口干、夜尿增多、容易疲劳等感觉；  
- 或计划调整饮食/运动但不确定是否适合自身情况。  

医生能结合您的整体状况，判断是否需要进一步检查（比如糖耐量、肝功能细化等），咱们不猜、不慌，用专业意见安心应对。
